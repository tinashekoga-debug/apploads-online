# AppLoads Icons

Required icon sizes for PWA installation:

- icon-72.png (72x72)
- icon-96.png (96x96) 
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192) - App icon
- icon-384.png (384x384)
- icon-512.png (512x512) - Splash screen

Also create:
- badge-72.png (72x72) - For notifications
- screenshot-mobile.png (375x667) - App store screenshot
- screenshot-desktop.png (1280x720) - App store screenshot

Design guidelines:
- Use the AppLoads brand color: #0b7d62
- Simple truck/loading icon design
- Ensure icons work on both light and dark backgrounds
- Make sure 192px and 512px icons are high quality
